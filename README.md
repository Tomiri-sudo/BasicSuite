# BasicSuite
Basic Programs Suite for Linux and Windows

Please note that all versions before 1.x.x are still considered in development and at the time are still mostly unusable.

## CHANGELOG!
```v0.0.0```<br>
<ul>
<li>Added first version of calculator. It's hardcoded strings are still in Italian (original language) so it needs translation later on.</li>
</ul>

<br>

```v0.0.1```
<ul>
<li>Calculator now has strings localized only in English.</li>
<li>Calculator now uses arrows after numeral inputs to not fill the terminal for no reason and to also keep things clean.</li>
</ul>

<br>

```v0.0.2```
<ul>
<li>Calculator now is compatible with Windows.</li>
<li>Now there is also a counter! That's also *mostly* cross-platform compatible, and most importantly, it does it's job in a clean way.</li>

</ul>

```v0.0.3```
<ul>
<li>Counter now is fully cross-compatible! the issue was that it failed to compile on linux because it tried to import libraries that are only available to Windows.</li>
<li>Now Counter has come to Android! While still pretty basic, it has dynamic theming, it can increase, decrease or reset your count, and it saves the count in memory so if you delete the cache or reboot, it won't delete your count.</li>
</ul>

```v0.0.4```
<ul>
  <li>Counter on Android now makes device vibrate when any of the buttons is clicked (UX improvement)</li>
</ul>
